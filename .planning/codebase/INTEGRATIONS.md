# Integrations

- **Database**: PostgreSQL via Prisma (Models configured in `prisma/schema.prisma`)
- **AI Providers**: Anthropic Claude & OpenAI SDKs installed (likely for AI Mock Interviews, resume analyzing, etc.)
- **Media Storage**: Cloudinary (profile pictures, resume uploads, etc., via `multer-storage-cloudinary`)
- **Payments**: Razorpay & Stripe (payment processing for premium features)
- **Mail/Notifications**: Nodemailer (SMTP based email delivery)
